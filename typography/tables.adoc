The following examples demonstrate some possible table layouts.

// https://mrhaki.blogspot.com/2014/11/awesome-asciidoctor-styling-columns-and.html

We can use the following styles:

- e: emphasized
- a: Asciidoc markup
- m: monospace
- h: header style, all column values are styled as header
- s: strong
- l: literal, text is shown in monospace font and line breaks are kept
- d: default
- v: verse, keeps line breaks

|===
